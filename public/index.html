Coolify Local Control

This page is informational only. Deployments and health data are managed in your Coolify dashboard.

What this project is

A small Node app to verify your local Coolify installation and provide a clean homepage for your test environment.

How to deploy updates

  1. Push code changes to your GitHub repository.
  2. Open the app in Coolify and click Redeploy, or configure Git webhooks.
  3. Watch build output in the Deployments and Logs tabs.
